CONDICIONES COMERCIALES

Con la finalidad de establecer reglas de negocio y minimizar los errores durante el proceso de facturación se definen listas de precios, plazos y planes comerciales que posibilitan:

✓ Uso de diferentes monedas: pesos, dólares, euros, etc.
✓ Control de fechas de vigencia.
✓ Asignación de precios por grupos de productos.
✓ Múltiples cuotas de vencimientos de comprobantes.
✓ Descuentos/recargos por montos o porcentuales, fijos y variables.
✓ Aplicación de descuentos/recargos sobre diferentes niveles.

FACTURACIÓN ELECTÓNICA

La emisión de comprobantes electrónicos está totalmente automatizada, estableciendo la conexión con AFIP al momento de solicitar autorización, y permite:

✓ Emisión de comprobantes tipo A, B y C (clientes locales).
✓ Emisión de comprobantes de exportación, tipo E.
✓ Impresión de comprobantes proforma.
✓ Visualización de impuestos calculados y monto final de comprobantes antes de solicitar autorización de AFIP.
✓ Obtención de CAE y número de comprobante on-line desde AFIP.
✓ Generación de comprobantes en pdf, para ser almacenado, impreso o enviado al cliente vía e-mail.
✓ En caso de no aprobación de un comprobante por parte de AFIP, se puede consultar el motivo del rechazo.
✓ Consulta de alícuotas on-line y cálculo de percepciones de Ingresos Brutos de ARBA y C.A.B.A.
✓ Generación de reportes y estadísticas de ventas.
✓ Calculo automático de impuestos y otros conceptos, ya sean fijos o variables, IVA, percepciones, descuentos, recargos, fletes, etc.
✓ Pantallas de ayuda interactivas durante el ingreso de datos al emitir comprobantes.

ADMINISTRACIÓN DE CLIENTES

Los datos de los clientes se ingresan y actualizan en el sistema para consultar la cuenta corriente y agilizar la emisión de los próximos comprobantes, permitiendo:

✓ Carga automática de datos de clientes mediante conexión on-line con AFIP.
✓ Asignación de lista de precios, plazo comercial y plan comercial por defecto.
✓ Porcentajes fijos de descuentos y recargos asignados a cada cliente.
✓ Segmentación de clientes por grupos con fines estadísticos.
✓ Almacenamiento de datos generales para utilizar como agenda.
✓ Administración de excepciones impositivas.

CUENTAS CORRIENTES

El estado de la cuenta corriente de cada cliente se administra desde este módulo, pudiendo realizar:

✓ Emisión de recibos por cobranza automáticos y manuales.
✓ Ingreso de comprobantes internos con fines de ajuste.
✓ Reportes de deuda vencida y a vencer.
✓ Consulta de comprobantes emitidos.

ESTADÍSTICAS

Se incluye un módulo de reportes que resulta ideal para el análisis de la gestión de ventas a nivel gerencial que permite generar:

✓ Ranking de productos
✓ Ranking de clientes
✓ Ranking de vendedores
✓ Ranking por zona de venta
✓ Ranking por categorías de productos
✓ Ranking por categorías de clientes

STOCK

Para quienes tienen la necesidad de administrar el stock de productos, el módulo de stock les permite:

✓ Uso de múltiples bodegas.
✓ Sencilla administración de stock por producto.
✓ Reserva de stock por producto.
✓ Selección de ítems administrables por producto, según la actividad de la empresa (color, talle, número de lote, despacho de importación, etc.)
✓ Consulta de ficha de stock por producto.
✓ Análisis de ventas y stock disponible para reposición de mercadería.

IMPUESTOS

✓ Libro de IVA ventas

✓ Ventas por actividad y provincia
✓ Posición mensual y anual

QUIÉNES SOMOS?

POLISOFT es una empresa dedicada a desarrollar soluciones informáticas apoyándose en las nuevas tecnologías y entendiendo que las mismas son fundamentales para dar solución a los desafíos que el mundo actual le requiere a las pequeñas, medianas y grandes organizaciones en un mercado cada vez más competitivo.

Nuestra experiencia de más de 20 años desarrollando soluciones para empresas multinacionales en diferentes países de América (Argentina, Uruguay, Chile, Brasil, Colombia y México) amplió nuestros conocimientos, permitiéndonos mejorar los procesos administrativos, comerciales y contables, maximizando el uso de los recursos.

Esa experiencia nos animó a aplicar los conocimientos obtenidos al mercado de pequeñas y medianas empresas que necesitan sistemas de información que acompañen el crecimiento de sus negocios sin descuidar su objetivo principal y contando con herramientas que les permitan tomar las decisiones más adecuadas para lograr una gestión exitosa. Los invitamos a conocernos. Estamos convencidos de que podemos trabajar juntos para lograr la solución óptima para su negocio.
